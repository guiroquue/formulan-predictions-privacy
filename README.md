# Privacy Policy – Formulan Predictions

Formulan Predictions does not collect, store, or share personal data on external servers.

All user data, including predictions, settings, and display names, is stored locally on the user’s device.

The app does not use analytics, tracking technologies, advertising services, or third-party data collection tools.

Notifications are generated locally on the device and are not associated with any remote service.
